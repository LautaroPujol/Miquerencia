# MIQUERENCIA
Este es un proyecto de ecommerce básico. La aplicación permite ver una lista de productos de ferretería y realizar compras en línea. Los productos y las órdenes de compra se almacenan en Firebase.

## Tecnologías Utilizadas:
- Vite
- React
- React-Router-DOM
- Firebase
- Toastify
- Bootstrap 

## Funcionalidades:

- Ver una lista de productos disponibles por stock
- Agregar productos al carrito de compras
- Ver el carrito de compras para chequearlo
- Realizar una orden de compra con los datos del usuario.
